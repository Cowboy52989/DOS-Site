Mission File Validation

Validate DayZ mission files for console safety

How Validation Works

The DOS Bot validates mission files in two stages:

  1. Stage 1: Format Validation - Checks JSON/XML syntax
  2. Stage 2: Content Validation - Verifies classnames and console compatibility
Supported formats: JSON, XML
üí° Pro Tip For best results, use the Discord bot command /upload followed by /validate. The bot provides detailed error reports and can limit results to prevent overload on large files.

üîç Validation System

The DOS Bot uses a two-stage validation process:

Stage 1: Format Validation

  • Checks if file is valid JSON or XML
  • Verifies structure matches DayZ format
  • Identifies syntax errors (trailing commas, missing brackets, etc.)

Stage 2: Content Validation

  • Validates classnames against database
  • Checks console-safe restrictions
  • Categorizes objects (Structures, Vehicles, Items, etc.)
  • Verifies event compatibility with maps
‚ö†Ô∏è Validation Limits Results are limited to 100 items per file to prevent overload. This ensures the bot doesn't freeze when processing large build files.

ü§ñ What is DOS?

DayZ Overwatch Sentinel (DOS) is a comprehensive Discord bot and web platform designed to help DayZ console server administrators validate mission files, manage GitHub repositories, and deploy files to Nitrado servers.

‚ú® Features

  • Mission File Validation - Two-stage validation for console safety
  • GitHub Integration - Automated pull request creation
  • Nitrado Deployment - Direct FTP upload to servers
  • Event Management - Apply seasonal decorations
  • Interactive Support - DOS-SUPPORT assistant
  • Web Dashboard - Complete control panel

üõ†Ô∏è Technology Stack

  • Backend: .NET 9.0, Discord.Net, Octokit
  • Frontend: HTML5, CSS3, JavaScript
  • Hosting: Railway (bot), GitHub Pages (site)
  • APIs: Discord, GitHub, Nitrado

üìù Console-Only Maps

DOS Bot supports the following console-exclusive DayZ maps:

  • Chernarus - Original DayZ map
  • Livonia - Dense forest environment
  • Sakhal - Arctic survival map

Modded maps are not supported due to console restrictions.
